# @babacoder/rn-page-size-helper

Package to check the page size of the app

## Installation


```sh
npm install @babacoder/rn-page-size-helper
```


## Usage


```js
import { multiply } from '@babacoder/rn-page-size-helper';

// ...

const result = multiply(3, 7);
```
